# enjinuity
Exports Enjin forums to other forum software using Selenium to scrape and parse
each page.

## Supported

### Forums
* MyBB
* phpBB

### Databases
* MySQL
* Postgres

## Prerequisites

### Required
* Python 3
* lxml
* Selenium

### Optional
* Psycopg2/PyMySql

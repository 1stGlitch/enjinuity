# enjinuity
Exports Enjin forums to other forum software using Selenium to scrape and parse
each page.

## Supported

### Forums
* MyBB
* phpBB

### Databases
* MySQL
* Postgres

## Install

### Prerequisites
* Python 3
* lxml
* Selenium
